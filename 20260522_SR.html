一、市场概况

2025/26榨季中国食糖产量创12年新高，工业库存升至历史高位，供需格局由紧转松。郑糖主力SR2609合约5月以来持续承压，围绕5400元/吨一线震荡。国际原糖受巴西丰产预期压制，但印度出口禁令和厄尔尼诺预期为远月提供支撑。

核心矛盾："弱现实+强预期"博弈——国内增产超预期、工业库存高位、销糖率偏低构成现实压制；但厄尔尼诺升温预期、印度出口禁令、国际油价高位支撑乙醇生产、成本底线等因素为远月提供托底支撑。短期缺乏突破性驱动，震荡为主。

4. 需求：消费平稳，替代品挤压 中性

2025/26年度食糖消费量预计1570万吨，与上年基本持平。淀粉糖等替代品持续挤压白糖消费空间，下游采购以随采随用为主，夏季消费旺季备货需求释放平稳但难超预期。市场谨慎心态上升，终端持糖价走低预期，难以带来糖价上涨驱动。

四、政策与宏观环境

1. 国内政策：进口管控+甘蔗保护价 偏多

  • 进口许可证管控：管理部门可能收紧进口许可证以稳定糖价、保护国内蔗农和糖厂利益
  • 甘蔗指导收购价：广西政府设定甘蔗收购指导价520元/吨，保障蔗农收益，构成成本底线
  • 糖浆管控：对糖浆及预拌粉进口管控趋严，但4月进口仍同比增长
  • 扶贫政策：蔗糖种植作为地区扶贫措施之一持续推动，种植面积稳中有增

2. 国际政策：印度出口禁令 偏多

  • 印度食糖出口禁令：5月中旬宣布立即生效，禁止原糖、白糖和精制糖出口至9月30日或另行通知
  • 禁令影响：印度是全球第二大糖生产国，出口禁令收紧全球贸易流供应，支撑国际糖价
  • 禁令性质：预防性举措，防止国内糖价飙升，但反映印度对供应的谨慎态度

3. 厄尔尼诺预期 偏多

  • ENSO报告：2026年4—6月期间可能进入厄尔尼诺状态，概率61%，并可能持续至年底
  • 影响路径：厄尔尼诺通常导致东南亚和印度干旱，影响甘蔗生长；巴西中南部则可能多雨影响收割
  • 市场影响：为远月合约提供天气溢价支撑，是"强预期"的核心来源

4. 能源价格与乙醇 偏多

  • 国际油价高位：美伊冲突推动原油维持100美元/桶附近，提升巴西乙醇生产收益
  • 制糖比偏低：巴西中南部4月制糖比仅32.93%，远低于正常48—51%水平，糖厂倾向产乙醇
  • 乙醇分流：高油价→乙醇收益提升→制糖比下降→食糖供应减少，构成利多支撑

⚠ 风险提示

  • 厄尔尼诺发展超预期：若厄尔尼诺强度超预期，可能导致印度、泰国、中国主产区干旱减产，推动糖价大幅反弹
  • 进口到港超预期：配额外进口许可已发放约260万吨，若集中到港将加剧国内供应压力
  • 巴西制糖比变化：若油价回落导致巴西糖厂提高制糖比，将增加全球食糖供应
  • 去库节奏不及预期：若夏季消费旺季需求疲软，工业库存去化缓慢，糖价承压时间延长
  • 宏观风险：美联储紧缩预期升温、美元走强压制大宗商品整体估值
  • 政策风险：进口许可证发放节奏、糖浆管控力度变化
  • 地缘冲突：美伊冲突推升油价→乙醇收益提升→制糖比下降→利多糖价（反之若缓和则利空）

八、综合研判与操作建议

基准情景（概率50%）：区间震荡，弱现实主导

国内增产超预期+库存高位压制糖价上方空间，但成本底线+政策托底+厄尔尼诺预期限制下方空间。郑糖SR2609在5300—5600元/吨区间震荡，ICE原糖在14—17美分/磅区间运行。去库节奏偏慢，夏季消费温和释放，无趋势性行情。

看多情景（概率25%）：厄尔尼诺+印度禁令驱动反弹

厄尔尼诺强度超预期，印度、泰国干旱加剧减产预期；巴西制糖比持续偏低；国内去库加速叠加进口管控收紧。郑糖突破5600元/吨上行至5800—6000区间，ICE原糖反弹至17—19美分/磅。下半年26/27榨季短缺预期升温推动价格中枢上移。

看空情景（概率25%）：供应压力集中释放，破位下行

进口糖集中到港+糖浆替代加速+夏季消费不及预期；巴西制糖比回升至50%以上；厄尔尼诺发展不及预期。工业库存去化缓慢，郑糖跌破5300元/吨成本线至5100—5200区间，ICE原糖跌破14美分/磅至12—13美分。

✔ 操作建议

  1. 短线操作：SR2609在5300—5600区间高抛低吸，5350以下轻仓试多（止损5280），5550以上轻仓短空（止损5620）
  2. 中线布局：5300元/吨附近为战略买入区（成本底线+政策托底），可分批建仓中线多单，仓位控制在总资金15%以内，博弈下半年厄尔尼诺+26/27榨季短缺驱动
  3. 套利策略：SR609-SR701价差-166元/吨，远月升水结构明确，可关注买远卖近的正套机会
  4. 期权策略：卖出SR2609看跌期权（行权价5300），收取权利金增强收益，同时买入看涨期权（行权价5600）博弈上行
  5. 对冲关注：若做多郑糖，需关注ICE原糖走势和人民币汇率变化对进口成本的影响
  6. 风险控制：单边仓位不超过总资金20%，严格止损；重点关注国内去库速度、巴西制糖比数据、厄尔尼诺发展
