SR白糖期货基本面分析

分析日期：2026年5月3日 | 预测区间：2026年5月—2026年10月

★ 核心结论（先看这里）

中期方向震荡偏弱，但下方空间有限
信心水平中等（多空因素交织，机构分歧较大）
SR2609合约价格区间5100—5800元/吨
核心建议不建议追空，也不建议重仓做多，以观望或轻仓区间操作为主

一、当前市场概况

截至2026年4月30日：

  • 郑糖主力合约SR2609收盘价约5406元/吨
  • 广西制糖集团现货报价5320—5410元/吨
  • 云南制糖集团报价5150—5230元/吨
  • 加工糖厂主流报价5610—5880元/吨
  • ICE原糖主力约14.5—15美分/磅
  • 配额内进口估算价3780—3810元/吨，配额外进口估算价4770—4820元/吨
通俗解释：白糖期货价格在5400元/吨附近，已经接近国内制糖成本线（5300—5500元/吨），继续下跌的空间不大，但上涨也缺乏强劲动力。

二、供应端分析 偏空

1. 国内增产明显

  • 2025/26榨季全国食糖产量预计约1250万吨（农业农村部4月预测），同比大幅增加，创十年新高
  • 广西产糖约740—780万吨，同比增加约94—130万吨
  • 云南产糖约213万吨，甜菜糖约120万吨
  • 5月后广西绝大部分糖厂收榨，云南5月底收榨，新增供应压力减弱
简单说：国产糖比去年多了很多，供应压力增大。但5月后进入纯去库阶段，新增供应利空出尽。

2. 进口量逐步放量

  • 配额内外进口利润窗口均已打开，配额外进口利润约970元/吨
  • 2026年1—3月累计进口52万吨，同比增加44万吨
  • 进口食糖许可证发放延迟，短期缓解供应压力，但后期放量预期强
  • 2025/26年度预计进口量约500万吨

3. 全球三大产糖国集体增产

国家2025/26榨季产量预估同比变化
巴西4100—4420万吨同比增长0.1%—1.6%
印度2900万吨以上同比增长约9.5%—20%
泰国约1200万吨同比增长约15%
  • 全球过剩：ISO预计过剩122万吨，Czarnikow预计过剩580—670万吨
  • 巴西4月上半月产糖64.7万吨，同比下降11.9%，短期因降雨压榨偏慢
  • 但Conab预计巴西全年产糖4420万吨，后期供应压力犹存
通俗解释：全世界都在产更多的糖，而吃糖的人没有明显增加，糖价自然承压。

三、需求端分析 中性偏弱

  • 国内食糖消费量约1570—1590万吨/年，增长缓慢（年增速约1%）
  • 4月为消费淡季，饮料、食品加工采购谨慎
  • 6月后进入消费旺季（中秋、国庆备货、夏季饮料），需求有望改善
  • 终端成品糖及软饮料产量同比增加明显，需求有所好转
  • 食品工业企业"随用随买"，不愿囤货，采购节奏偏慢
简单说：糖的供应在增加，但需求没有跟上。不过6月后进入旺季，需求端有望边际改善。

四、库存分析 偏空

指标数据解读
全国工业库存约710万吨创十年新高
广西工业库存约434—500万吨同比大幅增加
广西销糖率（3月底）41.38%同比-12.45%，去库偏慢
库存拐点预计5月中下旬6月起加速去库
  • 高库存意味着市场不缺糖，价格难有大涨
  • 5月中下旬库存见顶，6—9月旺季月均去库或达80万吨
  • 9月底库存或降至400万吨以下，进入偏紧区间
通俗解释：仓库里堆了很多糖，短期不缺货。但5月后库存开始下降，如果去库速度快，可能对价格形成支撑。

五、替代品分析 偏空

1. 淀粉糖（果葡糖浆）——白糖最核心的替代品

  • 2026年4月国内F55果葡糖浆市场价格约2735元/吨
  • 白糖现货价格约5300—5400元/吨，同等甜度换算后价差仍超2000元/吨
  • 当价差超过2000元/吨时，下游厂商替代意愿显著增强
  • 中国已成为全球最大淀粉糖生产国，2024年产量1450万吨，2026年预计达1663万吨
  • 果葡糖浆产能约900万吨，2026年新增产能约234万吨持续释放

2. 糖浆和预拌粉进口

  • 2026年1—2月糖浆与白砂糖预混粉进口14.22万吨，同比增加3.41万吨（增幅31%）
  • 2025/26榨季（10—2月）合计进口44.20万吨，同比减少30.54万吨（降幅41%）
  • 政策管控收紧是减少主因，但仍有流入，进口压力不容忽视

3. 替代关系总结

通俗解释：果葡糖浆是用玉米做的甜味剂，比白糖便宜很多。饮料厂、食品厂在白糖贵的时候会多用果葡糖浆代替白糖。现在白糖比果葡糖浆贵一倍多，很多工厂已经在替代了，这减少了白糖的需求。糖浆预拌粉虽然政策管得严了，但还是有进口，也在替代白糖。

六、宏观与政策因素 偏多支撑

1. 厄尔尼诺天气风险（最大变数！）

  • NOAA预测2026年5—7月厄尔尼诺发生概率61%—75%
  • 超强厄尔尼诺（海温超2℃）概率约25%
  • 厄尔尼诺导致印度、泰国干旱，甘蔗减产——利多糖价
  • 但厄尔尼诺给巴西带来更多降雨，有利于甘蔗生长——利空糖价
  • 6月后预测可靠性提高，届时方向更明确
通俗解释：厄尔尼诺是一种气候现象，可能让印度和泰国变干旱，甘蔗长不好，糖就少了，糖价可能涨。但巴西反而可能雨更多，糖更多。这是目前最大的不确定因素。

2. 原油价格

  • 原油维持80—85美元/桶高位运行
  • 油价高→乙醇利润高→巴西糖厂减少制糖比例→原糖供应减少
  • 2026/27榨季巴西制糖比可能从50.74%降至48%以下
  • 巴西汽油降价削弱乙醇竞争力，但整体制糖比仍有下降预期

3. 政策托底

  • 糖浆预拌粉进口政策收紧，限制替代品冲击
  • 糖价已跌至成本线附近，国家收储、进口配额调整等托底政策概率增大
  • 进口食糖许可证发放延迟，短期缓解供应压力

4. 地缘政治

  • 中东冲突影响全球航运，进口糖到港可能延迟
  • 尿素等化肥价格上涨，抬升甘蔗种植成本
  • 但地缘影响偏短期，糖价终将回归基本面

七、主要机构观点汇总

机构核心观点价格判断
永安期货糖价承压运行是大概率事件，价格中枢下移；关注进口排船及浆粉政策偏空
国泰君安期货震荡偏弱，内强外弱格局延续；定价锚定生产成本，关注内外价差收窄广西现货5200—5800元/吨
光大期货原糖增产价格弱势运行；阶段性利多来自天气；下有成本及政策支撑、上有高库存压力原糖13.5—19.5美分/磅
中信建投期货短期走势偏弱，进口压力压制盘面；厄尔尼诺风险抬升，或拉长熊市周期郑糖主力5100—5500元/吨
中信期货短期受中东冲突影响区间震荡；中长期关注厄尔尼诺，26/27榨季可能转向震荡偏弱

共识：各机构普遍认为中期偏空，但对下方空间有分歧，部分机构认为成本线附近有支撑。

分歧：厄尔尼诺的影响程度存在较大不确定性。永安、中信建投偏空，认为供应过剩主导；中信、光大认为厄尔尼诺可能改变中期格局；国泰君安认为内强外弱格局延续。

八、中期价格预测（2026年5月—10月）

情景一：基准情景（概率约55%）

价格区间：5100—5700元/吨

  • 厄尔尼诺影响有限或形成时间偏晚，全球增产兑现
  • 国内进口正常放量，去库节奏平稳
  • 糖价在成本线附近震荡，重心缓慢下移
  • 6月后旺季消费提供阶段性支撑
情景二：利多情景（概率约25%）

价格区间：5500—6200元/吨

  • 厄尔尼诺6月前确认形成且强度超预期，印度、泰国减产
  • 巴西制糖比大幅下降（降至48%以下），国际糖价反弹
  • 国内去库加速，旺季消费超预期
  • 进口政策进一步收紧
情景三：利空情景（概率约20%）

价格区间：4800—5300元/吨

  • 全球大幅过剩，进口大量涌入
  • 厄尔尼诺未兑现，巴西丰产超预期
  • 消费持续低迷，去库缓慢
  • 糖价跌破成本线，但政策托底限制下方空间

九、给您的建议

总体建议：谨慎观望为主

  1. 如果您目前没有持仓：建议先观望，等待更明确的信号。当前多空因素交织，方向不明确。
  2. 如果您想做交易：
    • 在5100—5200元/吨附近可考虑轻仓试多（接近成本支撑）
    • 在5700—5800元/吨附近可考虑轻仓试空（进口成本压制）
    • 严格控制仓位，不超过总资金的10%—20%
  3. 重点关注时间节点：
    • 5月中旬：云南收榨，库存拐点确认
    • 6—7月：厄尔尼诺是否兑现，巴西压榨进度，印度季风雨
    • 7—8月：国内夏季消费旺季，去库速度
    • 9—10月：新榨季开榨前，北半球估产情况
  4. 关键风险信号：
    • 如果厄尔尼诺确认形成且强度超预期 → 糖价可能快速反弹
    • 如果进口量突然大幅增加 → 糖价可能加速下跌
    • 如果国家出台收储政策 → 糖价底部确认

⚠ 风险提示（请务必注意）

  • 期货交易风险很大，可能出现比本金更大的亏损，请务必谨慎
  • 以上分析基于当前公开信息和机构研报，情况随时可能变化
  • 机构的研究报告也可能出错，过去判断正确不代表未来也对
  • 厄尔尼诺等天气因素预测难度大，实际影响可能与预期差别很大
  • 替代品（果葡糖浆等）持续扩产，长期压制白糖需求
  • 建议您在做任何投资决定前，咨询专业的金融顾问
  • 本分析不构成任何投资建议，仅供参考

总结

当前白糖市场"糖多、需求弱、库存高、替代品便宜"，中期价格大概率偏弱运行。但成本线附近有支撑，厄尔尼诺是最大的变数。建议您以观望为主，不要急于入场，等方向更清晰再做决定。

核心逻辑链：全球增产+国内增产+进口放量 → 供应充足 → 库存高企 → 价格承压；但成本线+政策托底+厄尔尼诺预期 → 下方空间有限。
