RB螺纹钢期货基本面分析

分析日期：2026年5月3日 | 预测区间：2026年5月—2026年10月

★ 核心结论（先看这里）

中期方向宽幅震荡偏弱，重心下移
信心水平中等（需求走弱确定性强，但成本支撑+政策托底限制跌幅）
RB2610合约价格区间2950—3400元/吨
核心建议反弹做空为主，不追空，关注成本支撑位

一、当前市场概况

截至2026年4月30日：

  • 螺纹钢主力合约RB2610收盘约3180元/吨，4月震荡偏强
  • 全国螺纹钢现货HRB400E 20mm均价约3360—3390元/吨
  • 螺纹钢周产量214.93万吨，同比下降6.2%
  • 螺纹钢周度表观消费量250.44万吨，同比下降3.7%
  • 螺纹钢总库存766.45万吨（社库579.1+厂库187.35），连续6周去化
  • 五大钢材品种总库存1702.56万吨，周环比降62.12万吨
  • 247家钢厂日均铁水产量239.32万吨，高位运行
  • 钢厂盈利率49.78%，月环比回升7.36个百分点
  • 华东长流程螺纹即期利润约90元/吨
  • 华东谷电利润约20元/吨
  • 1—3月累计出口钢材2471.7万吨，同比下降9.9%
  • 1—3月粗钢产量2.48亿吨，同比下降4.6%
通俗解释：螺纹钢目前处于"旺季不旺、淡季更弱"的困境。4月本应是钢材需求最旺的"金三银四"，但今年需求同比还是下降了8%以上。虽然库存连续6周下降，但主要靠的是减产（产量同比-6.2%），而不是需求好。钢厂刚刚恢复到一半盈利，生产意愿在增强，但5月以后南方梅雨季来了，需求会更弱。简单说就是"旺季快结束了，需求还没起来，供应又要增加"。

二、供应端分析 中性偏空

1. 粗钢产量——同比下降但环比回升

  • 2026年3月全国粗钢产量8704万吨，同比下降6.3%
  • 1—3月累计粗钢产量2.48亿吨，同比下降4.6%
  • 3月钢材产量13098万吨，同比下降2.3%
  • 螺纹钢周产量214.93万吨，同比下降6.2%，但环比已开始回升

2. 钢厂复产意愿——利润修复推动增产

  • 钢厂盈利率从低位回升至49.78%，近半数企业恢复盈利
  • 华东长流程利润约90元/吨，短流程谷电利润约20元/吨
  • 利润修复 → 钢厂复产意愿增强 → 供应压力增加
  • 日均铁水产量239.32万吨，处于年内高位
  • 5月预计日均铁水围绕241万吨高位波动，呈现"前高后稳"

3. 节能降碳政策——长期约束

  • 节能降碳政策对高耗能产能释放形成约束
  • 但短期对实际产量的限制力度有限
  • 更多影响长期产能结构和远月预期

4. 钢材出口——同比下降但仍有支撑

  • 1—3月累计出口钢材2471.7万吨，同比下降9.9%
  • 3月出口913.5万吨，环比增长16.6%
  • 出口均价684.7美元/吨，环比下跌6.1%
  • 海外需求复苏缓慢，出口难有持续性高增
  • 伊朗暂停钢坯和钢板出口至5月30日，短期减少国际供应
通俗解释：螺纹钢的供应端是"先减后增"的走势。一季度钢厂亏损严重，大量减产检修，所以产量同比下降6%以上。但4月钢厂利润开始恢复（近半数盈利了），钢厂就又开始增产了。5月以后供应还会继续增加，而需求却在走弱，这就是"供增需减"的不利局面。

三、需求端分析 偏空（核心压制）

1. 房地产——最大拖累，持续探底

  • 2026年1—4月螺纹钢总消费量2953.63万吨，同比下降8.56%
  • 4月单月螺纹钢消费量1171.59万吨，同比下降8.57%
  • 房地产用钢占比已降至15%以下，2026年预计同比再降7.7%
  • 中金预测：2026年房地产投资/新开工面积同比-14.9%/-16.1%
  • 地产用钢预计1.73亿吨，同比降低5.3%
  • 新开工面积持续萎缩，拖累建筑用钢需求
  • 4月建筑钢材日均成交量10.98万吨，同比下降5.5%

2. 基建——托底但力度有限

  • 超长期特别国债1.3万亿元，地方专项债4.4万亿元
  • "两重"建设资金累计安排6065亿元，占全年76%
  • 但4月专项债发行节奏明显放缓，仅完成当月计划的37%
  • 资金投放放缓预计5月中下旬传导至终端采购
  • 基建仅靠存量项目赶工托底，新增项目接续不足

3. 制造业——结构性亮点

  • 1—4月冷轧板卷消费量同比增长4.71%
  • 1—4月中厚板消费量同比增长6.08%
  • 汽车板、电工钢等高端板材消费量同比增长8%
  • 但制造业用钢以板材为主，对螺纹钢（建材）拉动有限

4. 季节性——5月后进入淡季

  • 5月下旬南方进入梅雨季，施工节奏回落
  • 7—8月高温天气进一步压制施工
  • "金三银四"旺季已过，需求季节性见顶
  • 4月的"韧性"更多是季节性环比修复，而非趋势性回暖
通俗解释：螺纹钢的需求端是当前最大的利空。螺纹钢主要用于盖房子和修路桥（建筑用钢占大头），但房地产还在持续下滑（新开工面积同比-16%），基建虽然政府在花钱但力度不够。更要命的是，5月以后南方要下梅雨了，工地干不了活，钢材需求会进一步下降。制造业（汽车、造船、风电）用钢在增长，但它们用的是板材，不是螺纹钢。简单说就是"房子盖得少、路桥修得慢、雨季又要来了"。

四、库存分析 中性偏多（短期）

1. 当前库存——连续去化但同比偏高

指标数据解读
螺纹钢总库存766.45万吨周环比-35.51万吨，去化加速
社会库存579.1万吨同比+13.86%，仍处高位
钢厂库存187.35万吨同比-3.3%，出货改善
五大品种总库存1702.56万吨连续6周去化

2. 库存趋势——去库节奏将放缓

  • 4月去库加速主要靠节前补库+旺季刚需，持续性存疑
  • 5月后需求季节性走弱，去库速度将明显放缓
  • 若供应增加+需求走弱，可能出现"去库放缓甚至重新累库"
  • 社会库存同比+13.86%，需求修复还没完全消化前期累库
通俗解释：库存虽然在下降，但主要靠的是"减产"而不是"多卖"。社会库存比去年同期还高了14%，说明仓库里的货还是太多。5月以后如果钢厂增产+需求减少，库存可能又堆起来。库存这个因素短期偏多（在去化），中期偏空（去化将放缓）。

五、替代品分析 中性

1. 型钢/H型钢——部分替代

  • 型钢和H型钢在建筑结构中可部分替代螺纹钢
  • 但替代比例有限（约5%—10%），且替代场景受限
  • 当前型钢与螺纹钢价差不大，替代动力不足

2. 装配式建筑——长期替代趋势

  • 装配式建筑（工厂预制构件）减少现场钢筋使用量
  • 政策大力推广装配式建筑，长期将减少螺纹钢需求
  • 但短期替代量有限，更多影响长期需求结构

3. 新国标——成本推升而非替代

  • 2024年新国标（GB/T 1499.1-2024）对螺纹钢强度、成分、工艺要求大幅提升
  • 小规格成本上浮80—150元/吨，大规格上浮150—250元/吨
  • 这不是替代品影响，而是成本推升，对价格有一定支撑
  • 新国标推动定价逻辑从"覆盖边际成本"转向"全面反映生产成本"

4. 板材对建材的替代——需求结构转换

  • 中国钢铁行业正从"以房地产为核心"转向"以制造业升级为驱动"
  • 建筑用螺纹钢消费量同比骤降15%，而汽车板、电工钢等高端板材增长8%
  • 这不是螺纹钢被替代，而是整体需求结构在转变——螺纹钢的需求占比在缩小
通俗解释：螺纹钢的替代品威胁不大（型钢替代有限，装配式建筑是长期趋势），但有一个更根本的问题——螺纹钢的"市场"在缩小。以前中国盖房子多，螺纹钢需求大；现在房子盖得少了，工厂和造船用钢多了，但那些用的是板材不是螺纹钢。所以螺纹钢面临的不只是"被替代"，而是"需求结构在转变"，这个趋势对螺纹钢是长期利空。

六、宏观与政策因素 多空交织

1. 稳增长政策——托底但力度有限

  • 政府工作报告：超长期特别国债1.3万亿元+地方专项债4.4万亿元
  • "两重"建设资金累计6065亿元，占全年76%，下达进度快于去年
  • 政治局4月28日会议：增强货币政策前瞻性灵活性，保持流动性充裕
  • 努力稳定房地产市场，扎实推进城市更新
  • 但政策传导到实际钢材需求需要时间，短期效果有限

2. 成本支撑——铁矿石+焦炭

  • 铁矿石价格高位震荡，日均铁水239万吨支撑原料需求
  • 焦炭价格相对稳定，钢厂原料库存偏低
  • 地缘政治推高油价和航运成本，对大宗商品价格带来支撑
  • 中金预计2026年铁矿年均衡价格90美元/吨，较2025年跌约12%
  • 若原料价格松动，成本支撑减弱，钢价下方空间打开

3. 中东地缘冲突——边际影响减弱

  • 前期地缘冲突推升油价和航运成本，间接支撑钢价
  • 但市场对中东局势关注度下降，对黑色板块扰动边际减弱
  • 伊朗暂停钢坯出口至5月30日，短期减少国际供应

4. 行业亏损——底部支撑

  • 一季度黑色金属冶炼和压延加工业利润总额亏损33.4亿元
  • 行业亏损意味着钢价已接近成本线，下方空间有限
  • 若钢价跌破成本线，钢厂将主动减产，形成底部支撑

八、中期价格预测（2026年5月—10月）

情景一：基准情景（概率约45%）

价格区间：3050—3300元/吨

  • 5—6月需求季节性走弱，南方梅雨季压制施工
  • 钢厂利润尚可，供应维持高位
  • 库存去化放缓，可能出现小幅累库
  • 成本支撑仍在（铁矿石+焦炭价格未大幅下跌）
  • 政策托底预期提供底部支撑，但缺乏上涨驱动
  • 钢价宽幅震荡，重心缓慢下移
情景二：利多情景（概率约20%）

价格区间：3250—3550元/吨

  • 专项债发行加速，基建资金到位，需求超预期回暖
  • 房地产政策大幅放松，新开工面积止跌回升
  • 节能降碳政策严格执行，粗钢产量大幅压减
  • 铁矿石价格持续高位，成本支撑强劲
  • 钢材出口超预期增长
情景三：利空情景（概率约35%）

价格区间：2800—3100元/吨

  • 房地产持续探底，新开工面积同比-20%以上
  • 梅雨季+高温天气双重压制，建材需求断崖式下滑
  • 钢厂复产过快，供应大幅增加
  • 铁矿石价格大幅下跌，成本支撑崩塌
  • 库存重新大幅累积，贸易商恐慌性抛售
  • 全球经济衰退，钢材出口大幅萎缩
注意：螺纹钢的利空情景概率设为35%（所有品种中最高），原因是需求端面临"房地产持续探底+季节性走弱+需求结构转变"三重压制，而供应端又在恢复，"供增需减"的格局较为确定。利多情景概率仅20%，因为需要政策大幅超预期才可能扭转颓势。

九、给您的建议

总体建议：反弹做空为主，不追空，关注成本支撑位

  1. 如果您目前没有持仓：
    • 建议在3250—3350元/吨附近轻仓试空
    • 止损设在3420元/吨以上
    • 第一目标看3100元/吨，第二目标看2950元/吨
  2. 如果您想做多：
    • ✔ 仅建议短线操作，在成本支撑位附近（2950—3050）轻仓试多
    • 止损设在2900元/吨以下
    • 目标看3150—3250元/吨，不宜贪多
    • 中期做多需等待政策超预期或库存大幅去化信号
  3. 如果您已持有空单：
    • ✔ 可继续持有，"供增需减"的中期逻辑没有改变
    • 在3050元/吨以下可考虑部分止盈，锁定利润
    • 关注成本支撑位，若铁矿石等原料价格大幅下跌可加仓
  4. 重点关注时间节点：
    • 5月上旬：节后需求恢复情况，去库是否延续
    • 5月中下旬：南方梅雨季来临，建材需求拐点
    • 6月：专项债发行是否加速，基建资金到位情况
    • 7—8月：高温淡季，库存是否重新累积
    • 9—10月：传统"金九银十"，需求是否季节性回暖
  5. 关键风险信号：
    • 📈 如果专项债发行大幅加速 → 基建需求可能超预期，空单需警惕
    • 📈 如果房地产政策大幅放松 → 需求可能企稳，空单考虑减仓
    • 📈 如果铁矿石价格暴跌 → 成本支撑崩塌，可加空
    • 📈 如果库存重新大幅累积 → 供增需减确认，可加空
    • 📈 如果钢厂大面积亏损减产 → 供应收缩，空单需止盈
    • 📈 如果节能降碳政策严格执行 → 产量受限，空单需谨慎

简单说：螺纹钢是所有已分析品种中唯一中期方向偏空的。核心原因就是房地产——螺纹钢是"盖房子用的钢"，而中国房地产还在持续下滑，这个趋势短期内无法扭转。其他品种要么供应紧缺（铜、铝、银），要么有政策利好（棉花、玉米），要么有金融属性支撑（金、银），但螺纹钢既没有供应紧缺，也没有强劲需求，更没有金融属性。螺纹钢唯一的支撑是"成本"（铁矿石和焦炭价格），如果成本塌了，螺纹钢就真的没什么能撑住了。

⚠ 风险提示（请务必注意）

  • 期货交易风险很大，杠杆效应可能放大亏损，请务必谨慎
  • 螺纹钢价格受政策影响极大，一项政策可能改变短期走势
